# basic-web-programming

This repository contains basic web programming learning materials for anyone who wants to start building websites. It covers HTML page structure, styling with CSS, and simple interactions using JavaScript.

## 🎯 What You Will Learn
Here are the topics covered in this repository.

### 🟧 HTML

### Basic HTML document structure 
- Common HTML tags: `<h1>`–`<h6>`, `<p>`, `<a>`, `<img>`, `<ul>`, `<ol>`, `<table>`
- Semantic HTML elements : `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>` 
- Building a website layout: header, navbar, sidebar, konten utama, footer 
- Using `<div>` for page structure

### 🟦 CSS

- Linking CSS to HTML 
- Selector, property, and value
- Styling text, colors, and backgrounds
- Box model: margin, padding, border The CSS Box Model
- Basic layout with Flexbox


### 🟨 JavaScript (Popup / Modal)


- Introduction to JavaScript in the browser 
- Simple DOM manipulation 
- Creating popups/modals with `onclick` and `style.display` 
- Basic event handling 


## 🛠️ How to Use

1. Clone this repo:
   ```bash
   git clone https://github.com/abindacarmo/basic-web-programming.git
   ```
2. Open the `prepara_esplikasaun/` folder to read the explanation materials.
3. Open the `preparasaun/` folder to see the full example project.
4. Open any `.html` file directly in your browser — no installation needed!
